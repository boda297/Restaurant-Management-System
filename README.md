# Restaurant Management System

A comprehensive Restaurant Management System built with Node.js, Express, and MongoDB.

## Features

- **Authentication**: Secure login and registration.
- **Inventory Management**: Track ingredients and stock levels.
- **Menu Management**: Add, update, and remove menu items.
- **Order Processing**: Handle customer orders efficiently.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/boda297/Restaurant-Management-System.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up your environment variables (e.g., `PORT`, `MONGO_URI`, `JWT_SECRET`).

4. Start the application:
   ```bash
   npm start
   ```
